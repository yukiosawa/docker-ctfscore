==================================
ctfscoreアプリケーションの作成メモ
==================================

(0) Windowsの場合、php.iniを編集してfileinfoを有効にしておくこと
(http://nl3.php.net/manual/ja/fileinfo.installation.php)
;extension=php_fileinfo.dll
のコメントを外す
extension=php_fileinfo.dll
そうしないと、添付ファイルダウンロード時にClass 'finfo' not found
エラーが発生する。


(1) DB設定を記述
APPPATH/config/db.php
APPPATH/config/development/db.php

(2) Simple Authを設定
cp PKGPATH/auth/config/auth.php APPPATH/config/.
必要箇所を編集
	'salt' => 'this_is_ctf_score_salt',

cp PKGPATH/auth/config/simpleauth.php APPPATH/config/.
必要箇所を編集
	'login_hash_salt' => 'this_is_ctf_score_salt',

APPPATH/config/config.phpを編集
-> always_loadでauthをロードするよう設定(コメントアウト)
	'always_load'  => array(
	         :
		'packages' => array(
		        'auth'
		),
		 :
	),

(3)
APPPATH/config/config.phpを編集

JQueryスクリプトを以下に配置
DOCROOT/assets/js/jquery-2.1.1.min.js

バリデーション時のエラーメッセージを日本語化
APPPATH/config/config.phpの該当箇所
'language'          => 'ja',
'language_fallback' => 'en',
'locale'            => 'ja_JP.UTF-8'
APPPATH/lang/ja/validation.php を配置


(必要があれば)APPPATH/config/ctfscore.phpを編集

各種ライブラリを以下に配置
- Chart.js http://www.chartjs.org/
 DOCROOT/assets/js/Chart.min.js
- animate.css http://daneden.github.io/animate.css/
 DOCROOT/assets/DOCROOT/assets/css/animate.css
- Lettering.js https://github.com/davatron5000/Lettering.js
 DOCROOT/assets/js/jquery.lettering-0.6.min.js
- textillate.js https://github.com/jschr/textillate
 DOCROOT/assets/js/jquery.textillate.js
